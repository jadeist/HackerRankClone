Temática del Hackaton

Consiste en un clone de hackerrank, donde cinta negra Front-end, Cinta Negra Backend y Cinta Roja unan sus fuerzas para lograr el objetivo.

Para esta versión de hackaton, todas las cintas tienen un rol específico que desempeñar.

Requisitos fundamentales

Todos los equipos deben ser lo más mixtos posibles, de esta manera hay gente de todas las cintas trabajando juntas.

Todos deben tener un Repo en github donde van subir el código.

En el Readme del repo deben escribir como hacer funcionar el Repo y como lo organizaron.

Los senseis anunciarán al equipo ganador por Slack el lunes 13 de noviembre.

Roles

Los roles a desempeñar son los siguientes:

  • - Los chicos de cinta negra Front-end se encargarán de hacer una interfaz donde se presentará el problema y recibirán la respuesta en un textbox, las respuestas no son en código solo lo que imprima el algoritmo
  • - Los chicos de cinta negra Backend se van a encargar o de crear endpoints donde de manera aleatoria impriman en formato Json los problemas o a través de templates
  • - Los chicos de cinta roja se encargar%aacute;n de construir los algoritmos que validen los inputs de los usuarios según el problema
